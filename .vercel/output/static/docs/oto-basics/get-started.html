OtO at a glance

OtO at a glance, Registration, update

This offline version may not be 100% up to date. Get the latest version at https://help.oto.software

  1. Toolbar
    • Toggles Prompting / Editing Mode
    • Provides access to the prompter screen, and to marker options and editing features  
      TIP

      If the Show Tooltips option is enabled in Settings Panel (Options Menu then ⚙ Settings, or 8 ), you can hover your mouse over any button in the interface to display a tooltip that will remind you of the button's associated function.

  2. Edition

    Editing window

  3. Rundown and documents

    Rundown management window (scripts list management)

  4. Mini Editor

    The mini editor allows you to view and scroll the prompting text without scrolling the prompter screen, and even edit it. You can also grab the panels handles to resize them.

  5. Markers module

    The marker module allows you to navigate between markers (similar to bookmarks within the text) and trigger advanced actions via actionMarkers.

  6. Search module

    Search and replace module

  7. Jump / Sync

    The "Jump" and "Sync" buttons allow you to respectively find a word or a selected portion of text from the main editing module to the mini-editor and vice versa.

  8. Settings

    Settings Panel (Options Menu then Settings, or 8 ) panel.

    To distinguish between the Menus (in the top bar of your computer) and the Settings (the gear icon button on the top right angle of OtO's user interface), this documentation will consistently refer to Menu and ⚙ Settings.

# Recommanded specifications

  • For PC:

    • Operating system: Windows 10, version 1607
    • CPU: Recent x86-64 processor (Intel/AMD)
    • RAM: 8+GB (16gb recommended)
    • GPU (optional for faster performance): NVIDIA GPU with CUDA support (e.g., GTX 1050 or better)
    • Storage: SSD (for faster model loading)

  • For Mac:

    • Operating system: macOS 10.15 (Catalina)
    • CPU: Apple Silicon (M1/M2) or Intel Mac
    • RAM: 8GB
